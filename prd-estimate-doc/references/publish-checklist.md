# Publish Checklist

## 打包前

- `SKILL.md` 中的描述只写“何时使用”，不要把执行流程塞进 description
- `agents/openai.yaml` 存在，且显示名、描述、默认提示词可读
- 所有引用路径都是相对路径，可在 skill 根目录内解析
- 没有真实 `baseToken`、真实表 ID、真实用户数据文件
- 没有临时文件，例如 `.DS_Store`、导出缓存、调试日志

## 结构检查

- `references/` 下只保留通用说明，不写项目私有信息
- `templates/` 提供最小可运行配置示例
- `scripts/` 的参数和错误提示完整
- 脚本默认行为优先只读预览，真正写入需要显式参数触发

## 使用检查

- 只读场景下，能根据 PRD 正常产出 4 份 markdown 文档
- Base 场景下，能根据模板配置运行脚本
- 叶子任务工时重写后，父任务工时为空
- AI 排期字段写入后，叶子任务都有 `AI直接实现 / 执行方式 / 执行波次`

## 交付检查

- README 说明了用途、安装方式、典型触发语句
- 脚本用途和配置方式已在文档中说明
- 版本号已更新
- 如需团队内分发，建议附一份示例 PRD 或演示截图
